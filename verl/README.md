# Overview

[Verl](https://github.com/verl-project/verl) is a popular reinforcement learning framework. 
This example demonstrate how to use Verl to perform supervised finetuning on Gemma 7b
model using SFT trainer.

# Git clone Verl

```bash
git clone https://github.com/verl-project/verl.git .
```

# How to run

```bash
sgcli run -f workload.yaml --watch 
```
